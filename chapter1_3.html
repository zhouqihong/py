1.3 PyCharm与Python的安装

1.3.1 PyCharm简介

PyCharm是JetBrains公司开发的一款强大的Python集成开发环境(IDE)，为Python开发提供了丰富的功能和工具。

PyCharm Community

免费

专为Python纯开发设计，完全免费且开源。

  • 智能代码编辑器
  • 代码导航和自动补全
  • 调试器和测试运行器
  • 版本控制集成
  • 科学工具集成

PyCharm Professional

付费

为专业开发者提供全栈开发工具。

  • Web开发框架支持
  • 数据库工具
  • 远程开发功能
  • 科学模式增强
  • 更多调试和分析工具
提示： 对于学习和数据科学应用，PyCharm Community版本完全足够，而且是免费的。本指南将使用Community版本进行说明。

1.3.2 PyCharm安装步骤

下载PyCharm Community

  1. 访问 PyCharm官网下载页面
  2. 选择您的操作系统（Windows、macOS或Linux）
  3. 点击Community版本的下载按钮

安装过程

  1. 运行下载的安装程序
  2. 选择安装位置（建议使用默认位置）
  3. 在安装选项页面，建议勾选以下选项：
    • 创建桌面快捷方式
    • 将PyCharm添加到PATH环境变量
    • 将.py文件与PyCharm关联
  4. 点击"Install"开始安装
  5. 安装完成后，启动PyCharm
提示： 首次启动PyCharm时，您可以选择主题（深色或浅色）和安装必要的插件。大多数情况下，使用默认设置即可。

1.3.3 Python安装步骤

下载Python

  1. 访问 Python官网下载页面
  2. 点击下载最新版本的Python（推荐Python 3.9或更高版本）

安装过程

  1. 运行下载的Python安装程序
  2. 在第一个安装界面，务必勾选"Add Python to PATH"选项
  3. 选择"Install Now"进行标准安装，或选择"Customize installation"进行自定义安装
  4. 等待安装完成
  5. 安装完成后，点击"Close"
重要： 安装Python时一定要勾选"Add Python to PATH"选项，否则您将无法在命令行中直接运行Python。

验证安装

打开命令提示符（Windows）或终端（macOS/Linux），输入以下命令：

python --version

如果显示Python版本号，说明安装成功。

1.3.4 配置PyCharm使用Python解释器

创建新项目

  1. 启动PyCharm
  2. 点击"New Project"
  3. 输入项目名称和位置
  4. 在"Base interpreter"部分，确保选择了您安装的Python版本
  5. 点击"Create"

检查Python解释器

  1. 转到 File > Settings (Windows) 或 PyCharm > Preferences (macOS)
  2. 导航到 Project > Python Interpreter
  3. 确保选择了正确的Python解释器
  4. 如果需要更改，点击下拉菜单选择其他解释器
提示： 如果PyCharm没有自动检测到Python解释器，您可以手动添加。点击设置图标，选择"Add"，然后浏览到Python安装目录。

1.3.5 创建第一个Python程序

创建Python文件

  1. 在项目视图中右键单击项目名称
  2. 选择 New > Python File
  3. 输入文件名（如"hello_world"）

编写和运行代码

# 这是一个简单的Python程序
print("Hello, World!")

# 计算并打印结果
x = 5
y = 3
result = x + y
print(f"{x} + {y} = {result}")

要运行程序，右键单击编辑器区域，选择"Run '文件名'"或使用快捷键 Ctrl+Shift+F10 (Windows/Linux) 或 Control+Shift+R (macOS)。

Ctrl+Y
删除当前行
Ctrl+F

Ctrl+R

Alt+Enter
