第一章 · Python简介与工具安装

Python数据科学入门

Python是当今最受欢迎的数据科学编程语言之一，以其简洁的语法、丰富的库和活跃的社区而闻名。本章将引导您完成Python环境的搭建和基本工具的使用，为后续的数据分析工作做好准备。

学习目标：完成本章后，您将能够搭建Python开发环境，了解不同开发工具的特点，并运行您的第一个Python程序。

本章内容概览

  • Python语言简介与数据科学生态系统
  • Jupyter Notebook的安装与使用
  • PyCharm社区版（专业版）的安装与配置
  • 云平台（Cloud Studio等）的使用方法
  • 编写并运行您的第一个Python程序

环境配置示例

使用Conda创建Python环境：

# 创建名为pyda的Python环境，版本为3.11
conda create -n pyda python=3.11

# 激活环境
conda activate pyda

# 安装必要的数据科学包
conda install numpy pandas matplotlib jupyter

第一个Python程序

# 问候世界
print("Hello, Python数据科学!")

# 简单计算
a = 5
b = 7
print(f"{a} + {b} = {a + b}")

# 使用Python进行数据分析
import numpy as np
data = np.array([1, 3, 5, 7, 9])
print(f"数据平均值: {np.mean(data):.2f}")
