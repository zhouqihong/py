Python常见语句

掌握Python中的常见语句是编写复杂程序的关键。本章将深入讲解条件语句、循环语句、用户输入与交互以及异常处理语句，帮助您构建更加健壮和交互性强的Python程序。

学习目标：完成本章后，您将能够熟练使用条件语句进行逻辑判断，掌握循环语句处理重复任务，理解用户输入与交互的方法，并能够编写健壮的异常处理代码。

本章内容概览

  • 条件语句：if、elif、else的使用
  • 循环语句：for循环和while循环
  • 用户输入与交互：input()函数的使用
  • 异常处理：try、except、finally语句

常见语句示例

# 条件语句示例
score = 85
if score >= 90:
    grade = "优秀"
elif score >= 80:
    grade = "良好"
elif score >= 60:
    grade = "及格"
else:
    grade = "不及格"

print(f"85分的成绩等级: {grade}")

# 循环语句示例
print("\nfor循环示例:")
for i in range(1, 6):
    print(f"当前数字: {i}")

print("\nwhile循环示例:")
count = 5
while count > 0:
    print(f"倒计时: {count}")
    count -= 1

# 用户输入与交互示例
print("\n用户输入与交互示例:")
name = input("请输入您的姓名: ")
age = input("请输入您的年龄: ")
print(f"您好，{name}！您今年{age}岁。")

# 异常处理示例
print("\n异常处理示例:")
try:
    num = 2  # 这里可以尝试修改为0或非数字值来测试异常
    result = 10 / num
    print(f"10除以{num}的结果是: {result}")
except ZeroDivisionError:
    print("错误: 不能除以零!")
except ValueError:
    print("错误: 请输入有效的数字!")
finally:
    print("程序执行完毕。")
